# Training OpenZeppelin

This is tutorial and practice repository to use OpenZeppelin

## Environments

- Node.js ... 16.10.0
- [OpenZeppelin](https://docs.openzeppelin.com/openzeppelin/) ... 4.3.2

## How to compile Solidity

```sh
$ npx hardhat compile
```
